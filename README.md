# Customer Segmentation using KMeans

## Objective
Segment customers based on income and spending behavior.

## Process
- Data cleaning
- Feature scaling
- Elbow method for K selection
- KMeans clustering
- Cluster visualization
- Business interpretation

## Optimal Clusters
K = 5

## Business Insight
Identified premium, budget, impulse, and conservative customer groups for targeted marketing.
